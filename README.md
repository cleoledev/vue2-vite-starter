# Vue 2 + Vite Legacy

## Implementation
- Vue 2
- Vite Legacy
- Vue Router
- Vuex
- ESlint
- Stylelint
- Autoprefixer
- Scss Toolkit

## Recommended IDE Setup

[VSCode](https://code.visualstudio.com/) with following extensions:
- johnsoncodehk.volar
- octref.vetur
- dbaeumer.vscode-eslint
- stylelint.vscode-stylelint

## Usage
### Development
```
npm run dev
```
### Production
```
npm run build
```
